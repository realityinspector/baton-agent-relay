# Color palette (current, 2026)

## Primary
- **Indigo 900** `#1A1B4B` — primary text on light, dark surfaces
- **Aurora 500** `#2EE6A6` — primary CTA, single accent

## Secondary
- **Slate 700** `#2A3243` — body text in dark mode, captions
- **Mist 050** `#F5F7FA` — page background light mode

## Functional
- **Success 500** `#1FA268` — confirms, completed states
- **Warning 500** `#E6A02E` — cautions, expiring tasks
- **Danger 500**  `#D13B3B` — destructive actions, errors

## Rules
- Aurora 500 is the **only** accent. Don't introduce a second.
- Primary CTA fills with Aurora 500, text Indigo 900.
- Don't tint Aurora 500. Use it at 100% or omit it.
- Body text: Indigo 900 on light, Mist 050 on dark.
- Functional colors are never used as decoration — only on functional UI.
