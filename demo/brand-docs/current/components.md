# Component do/don't (current, 2026)

## Buttons
- **Primary CTA:** bg Aurora 500, text Indigo 900, 12px radius, 16/24 Inter 600.
- **Secondary:** bg transparent, border 1.5 Indigo 900, text Indigo 900.
- **Destructive:** text Danger 500, no fill, no border (link-styled in red).

Don't:
- Don't use a filled red button. Destructive is text-only.
- Don't use shadows on buttons.
- Don't use Aurora 500 for any non-primary button.

## Inputs
- Border: 1px Slate 700 / 30%
- Focus: border 2px Aurora 500, no shadow
- Label: 13/18 Inter 500 above the input. Don't use placeholder as label.

## Forms
- Group related inputs visually with 8px gaps; unrelated with 24px.
- Errors inline below input, Danger 500 text, no icon.

## Tables (money)
- Right-align money columns. Tabular numerals.
- Negative values prefix with "−" (en-dash), not parens.
