# Typography (current, 2026)

## Faces
- **UI:** Inter (variable weight). Default body 14/20.
- **Marketing headlines:** Source Serif 4 (variable). Use weight 600+.

## Numerals
Tabular figures everywhere money is shown:
```css
font-variant-numeric: tabular-nums;
```

## Scale
