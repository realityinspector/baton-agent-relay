# Voice & tone (current, 2026)

## Voice (always)
- **Specific.** "$432 due Aug 15" not "money due soon."
- **Confident, not cocky.** "We compute self-employment tax on actual receipts, not estimates."
- **Plain.** "withholding" not "tax obligation accrual."
- **Imperative when prompting action.** "Add an account" not "You can add an account."

## Tone (varies by context)
- **Setup / onboarding:** warm, brief, action-first.
- **Money state:** neutral, precise.
- **Errors:** state what happened, what to do, never apologize.

## Don't
- Don't use exclamation points except for explicit success ("Filed!").
- Don't use "we're sorry" in errors. State the fix.
- Don't address the user as "friend", "buddy", "you got this".
- Don't use "magic", "delight", "smart" — these are vague.

## Examples (good)
- "Quarterly estimate due Sep 15: $1,847"
- "Connect a bank to start tracking. Takes ~30 seconds."
- "We couldn't reach Plaid. Try again or skip; you can add it later."
